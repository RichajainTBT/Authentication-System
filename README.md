# Authentication System (Login / Register)

##  Description
This is a simple front-end authentication system built using JavaScript and localStorage.  
It allows users to register, log in, and reset their password with proper form validation and user feedback.

---

##  Features
- User Registration with validation
- Login functionality with credential checking
- Forgot Password (reset password)
- Real-time + Blur-based form validation
- Confirm password matching
- Toast notifications for user feedback
- Duplicate email check
- Data persistence using localStorage
- Conditional rendering (Login / Register / Forgot)

---

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 5
